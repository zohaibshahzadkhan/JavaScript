Multiple line comments

Code 

		/*
		document.write("Hello world");
		document.write("Hello Pakistan");
		document.write("Hello Karachi");
		*/

Description 

	For multiple line comments in java script you have to use  /*  before a line you want to comment and */ after the line till where you want your comment to be .
        Example  
	        /*
		document.write("Hello world");
		document.write("Hello Pakistan");
		document.write("Hello Karachi");
		*/
