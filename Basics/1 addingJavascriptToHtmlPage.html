Adding java script to HTML page

Code 


	

		< head>
		< title>Java Script
		< script type="text/javascript">
	
		< /script>
		< /head>
		
		NOTE :  Do not give blank spaces inside tags i just do it to make Code visible on web page.

Description 

	In code section, I add javascript by using script tags between head  tags
	you can also write Script tags between body but the best practise is to write it between the head tags.
