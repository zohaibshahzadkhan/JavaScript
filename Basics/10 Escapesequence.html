Escape Sequence

Code 

         Single quote example
         document.write("Welcome to \'javascript\' world");
         
         Double quote example.
         Double quote ('Welcome to \"javascript\" world');

Description 

	Escape Sequence are present in javascript which can be used with string values . For example if you want to use Single Quote with string
        for this you have to write string in double quotes and than use Slash with single quote for the word you want to be in single quote .
        
        
        if you want to use double Quote with string
        for this you have to write string in single quotes and than use Slash with double quote for the word you want to be in double quote .
